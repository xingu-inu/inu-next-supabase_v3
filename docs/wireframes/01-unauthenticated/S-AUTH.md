**Visual Spec:**

| Element | Spec |
| ------- | ---- |
| Background | Ocean Gradient (Ocean-600 → Ocean-400, 135deg) |
| Card | Glass-3 (backdrop-blur: 16px, bg: white/60%, border: white/30%) |
| 🦭 Logo | 48px, 카드 상단 외부에 배치 |
| "INU" 텍스트 | h2, --text-primary, 로고 바로 아래 |
| 태그라인 | h3, --text-primary, 카드 내 최상단 |
| 안심 문구 | body, --text-secondary |
| Google 버튼 | Full-width, --bg-primary, 48px height, border-radius: 12px |
| 법적 링크 | caption, --text-tertiary, 카드 하단 외부 |

**Content Order (top → bottom):**

1. 🦭 (48px) + "INU" — 카드 외부, 중앙 정렬
2. 태그라인: "나를 알아가는 가장 쉬운 방법"
3. 안심 문구: "3초면 시작할 수 있어요"
4. [G Google로 계속하기] 버튼
5. 이용약관 · 개인정보처리방침 — 카드 외부, 중앙 정렬

**Interactions:**

| Click Target | Behavior |
| ------------ | -------- |
| [Google로 계속하기] | Supabase Google OAuth → 최초 방문 시 S-WEL, 재방문 시 S-CARD |
| 이용약관 | 이용약관 페이지 (외부 링크) |
| 개인정보처리방침 | 개인정보처리방침 페이지 (외부 링크) |
