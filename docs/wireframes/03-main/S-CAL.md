**Calendar View Rules:**

| Element | Description |
| ---- | ---- |
| Today highlight | `┏━━━┓` bold border to indicate today's date |
| Action chip | `╭──────╮│💪러닝│╰──────╯` category-colored chip |
| Monthly view | Shows Actions by date (category icon + name) |
| Weekly view | Detailed time-slot display (includes time axis) |
| Action click | Opens Goal detail modal (S-DETAIL) |
| Empty date click | Quick-add an Action on that date |
