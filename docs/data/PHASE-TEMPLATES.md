# Rule-Based Phase/Action Templates

> Rule-based templates for auto-generating a roadmap without AI when a Goal is created.
> 3 Phases per Category + WHAT combination, each Phase with 3 Actions + 1 coaching message.

---

## Status Mapping Rules

| Status | Starting Phase | Phases Generated |
| ------ | -------------- | ---------------- |
| `just_starting` (이제 시작할래요) | Phase 1 | All 1, 2, 3 generated |
| `some_experience` (조금 해봤어요) | Phase 2 | Only 2, 3 generated |
| `actively_doing` (열심히 하는 중) | Phase 3 | Only 3 generated |
